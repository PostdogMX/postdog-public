# Postdog 🐾

**Postdog** is a Mexican shipping aggregator platform that allows small and medium-sized e-commerce businesses to compare rates, generate shipping labels, and manage shipments from multiple carriers in one place.

## 🌐 Links
- Website: [postdog.com.mx](https://postdog.com.mx)
- App: [app.postdog.com.mx](https://app.postdog.com.mx)

## 📦 What we do
- Compare shipping rates from DHL, FedEx, Estafeta, J&T and more
- Generate shipping labels instantly
- Sync orders from WooCommerce, Shopify and other platforms
- Manage pickups and track shipments in one dashboard
